Itálico / Ênfase

Aqui temos um termo em itálico sem semântica com a tag i de italic

Aqui temos um termo com semântica usando a tag em de emphasis

Texto Marcado

Podemos criar um texto marcado usando a tag mark

E para mudar a cor da marcação usace o estilo backgound-color na tag

Texto Grande e Pequeno

Podemos criar um texto grande usando a tag big, mas já é uma tag obsoleta

Podemos criar um texto pequeno com a tag small

Textos Deletados e Inseridos

Podemos indicar um texto deletado com a tag del. Isso serve para indicar que um texto deve ser lido mas não considerado

Podemos indicar um texto inserido com a tag ins

Texto Sobrescrito e Subescrito

Para inserir um texto sobrescrito usa-se a tag sup. Exemplo x2

Para inserir um texto sobescrito usa-se a tag sub. Como: H2O

Trecho de código

Podemos inserir trechos de código com a tag code, como o exemplo: 


            Primeiramente inseri uma parte desse HTML aqui, mas o browser interpretou tudo como se fosse HTML mesmo, apenas mudando para monospace, e sem quebra de linha também.
            Depois descobri que existe a tag pre para manter a formatacao do texto como qubra de linha

Citações

Para inserir citações usa-se a tag q (de quote em ingles) dentro do parágrafo. Para criar citações separadas usa-se a tag blockquote, como essa:

Lorem ipsum dolor sit amet consectetur, adipisicing elit. Ipsam vero accusantium consectetur, sapiente maiores perspiciatis enim itaque unde suscipit beatae et, nesciunt minima culpa, molestiae tenetur quisquam nisi saepe magnam!

E para inserir a fonte usa-se o parametro cite dentro da tag

Abreviações

É possivel inserir abreviações com seus significados no HTML usando a tag abbr, isso faz com que o significado apareça quando o cursor passa pela abreviação
